# COMP380-Course-Project  
Brief text difficulty grader and reading aid AI  

TO RUN Locally:  
git clone https://github.com/JacobThmpsn/COMP380-Course-Project.git  
cd COMP380-Course-Project  
python3 -m venv .venv  
source .venv/bin/activate  
pip install -r requirements.txt  
streamlit run ui_app.py  

TO RUN Online:
https://jthompson-comp380-project.streamlit.app/
